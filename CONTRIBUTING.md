# Contributing to ASkyl.Dsm.WebHosting

Thank you for your interest in improving this project! To optimize development, please note our current contribution policy.

## ⚠️ Important: Scope of Contributions

At this stage of development, **external contributions are strictly limited to translations and localization**.

* **Code & Features:** Pull Requests containing code modifications, core feature developments, or bug fixes will not be accepted and will be closed. If you find a bug or have a feature request, please open an **Issue** instead.
* **Translations:** Pull Requests targeting new languages or improving existing localization files are highly welcome.

## 🌐 How to Contribute a Translation

1. Fork the repository.
2. Create a dedicated branch for your translation changes (e.g., `localization/add-french`).
3. Update or add the relevant localization/resource files.
4. Submit a Pull Request with a clear description of the targeted languages.

By contributing, you agree that your localization work will be licensed under the project's current GNU GPLv3 License.
