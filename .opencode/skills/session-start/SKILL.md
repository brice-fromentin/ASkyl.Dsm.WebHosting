# Session Start Protocol

## What I Do

At the START of EVERY SESSION, I MUST:

1. **FOLLOW** session start protocol described in instructions

## When to Use Me

- **ALWAYS** at session start - before any other action
- When user sends first message in a new session
- BEFORE processing any work requests

## Enforcement Rules

- DO NOT proceed with ANY work until protocol is complete
- DO NOT skip steps under any circumstances
- If user sends greeting FIRST, still complete protocol before doing work
- Log completion of protocol sequence
